## Installation
1. **Clone the Repository**
    ```bash
    git clone https://github.com/lagentiti/twitch-spam-for-ban-account.git
    ```
2. **Download Modules**
    ```bash
    npm i
    ```
## Use Project

1. **start server**
    ```bash
    node server.js
    ```
2. **start project**
    ```bash
    node index.js
    ```
